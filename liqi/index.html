Windows

  • 日常

    • Ditto：记录历史复制内容，方便粘贴。
    • StrokeIt：Windows鼠标手势，操作畅快淋漓。
    • Clean Master：想清理系统垃圾又不想装360，CM帮你，猎豹出品。该产品针对国际用户，还算良心。
    • Everything：极速搜索，快到发指。
    • FinalData：数据删除恢复利器。
    • KeyTweak键盘修改器：可以修改键盘按键。
    • USB Safely Remove：U盘退出一直被占用，又不想装360，就用它。
    • Wox：类似Mac中的alfred，你想要什么，立刻就有什么。
    • DailyPic：如果你用windows 10，如果你喜欢Bing的每日壁纸，这个应用可以设置Bing壁纸为windows 10的桌面壁纸和锁屏壁纸，非常不错。

  • 办公文档

    • ABBYY FineReader：目前用过的最强大的OCR文字识别软件，没有之一。
    • FileZilla：好用的FTP工具。
    • Nordri Tools：一款让你的PPT变的更强大的工具。
    • OneNote：我也用过印象笔记、有道云笔记、为知笔记，都很好，但是似乎OneNote越来越给力了。
    • SketchUp：想画个草图又不会用CAD，用它，不会错。
    • SmallPDF：有了它，各种PDF相关的软件都可以不用了。
    • PPT遥控器：百度出的，本来用起来心存疑虑怕来个全家桶，试用之后还不错吧。
    • ZoomIt：演示时可以临时缩放屏幕。

  • 音视频及图片

    • Camtasia Studio：目前用过最好的屏幕录制软件，后期处理实在是方便、好用、强大！
    • GoldWave：好用音频处理软件，比CoolEdit好用。
    • PotPlayer：高清播放器，可以配合 Zune 皮肤。
    • 图片批处理：很小、很老的一个软件，但是超级好用。还有这个BenVista PhotoZoom
    • 曦力音视频转换：视频格式转换能力超一流，不过貌似中文官网没了。

  • 网络相关

    • Proxifier PE：网络代理神器！
    • Shadowsocks：FQ神器，不过得自建VPS，或者买账号，配合Proxy SwitchyOmega，上网畅通无阻。

Develop

  • Sublime：前端工具精品，我正在用它写这些文字。
  • Chrome：
    • Advanced REST client：测试各种访问请求。
    • Isometric Contributions：让Github的提交次数以可视化的方式展现。
    • JSONView：自动格式化JSON数据。
    • Octotree：以树状列表的方式浏览Github代码。

iPhone

  • 快喵：速度快、无广告、不免费的翻墙VPN，貌似美国区的AppleStore才能下载。
  • IFTTT：If this, than that.只要满足某个条件，则执行一个动作。比如快要下雨了提醒你带伞、某个人发了微博提醒你、每天给自己发一封邮件等。
  • Kindle：没买Kindle，也可以看电子书，有了它，喜欢上看电子书了，买了不少。

Chrome

  • Proxy SwitchyOmega：强烈推荐！管理你的代理网络，配合自动切换选项，上网畅通无阻！
  • Page Monitor：如果你想后台监测一个网页是否更新，比如公司或者学校是否发布新消息，可以用它定时监测。
  • Ghostery：屏蔽广告、防止追踪、保护隐私，用它，离不开它。
  • 一键切换：在各个搜索引擎间一键切换搜索。
  • iCloud 书签：让你的Safari与Chrome书签同步，在手机上方便继续浏览。貌似需要与iCloud Panel配合。
  • OneTab：打开了一大堆页面，突然有事又不舍得关？它来帮你。
  • Replace Google CDN：加速某些国外网站的访问。
  • RSS Feed Reader：如果你还在用RSS..
  • Video Downloader professional：网页视频下载神器。
  • 购物党全网自动比价工具：唯一推荐的一个购物比价。

Web Site

  • 阿里妈妈：淘宝买东西返利给自己。
  • Google照片：简直神器！！配合手机app，自动上传备份照片、自动编辑相册、自动识别人物、自动生成相册影片… 全部自动！
  • ProcessOn：在线协作的Vision+Axure+JustinMind，挺不错的。
  • 百度图说：做一个简单的统计图表展示，还是不错的，还可以交互，UI方面有待提升。
  • 百度脑图：一口气推荐了好几个百度的东西，确实都还不错。

科研利器

  • sci-hub：它可以让你畅通无阻的使用Google Scholar，又可以畅通无阻的下载论文！

设备

  • iPad mini：就像当年买小霸王一样，打着学习的接口买了个游戏机、播放器 -_-#
